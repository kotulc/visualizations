Command Line Options
====================

These flags allow you to generate sample plots from the base subnet_plot module.
See :doc:`examples` for some basic use cases.

.. option:: -a <bool>, --animate <bool>

    Enable or disable plot animation.
    Value is either 0 or 1.

    Default: 1 [True]
    
.. option:: -s <int>, --steps <int>

    Render each sample plot with the given number of randomly generated data points.

    Default: 25
