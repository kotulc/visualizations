Visualizations Documentation
============================

*Visualizations* is a simple framework for combining and animating Matplotlib subplots.

This documentation is intended to help you get started using and extending this framework.

Before using this framework, it is strongly suggested that you start by familiarizing yourself with Matplotlib's pyplot interface using this introductory `tutorial. <https://matplotlib.org/3.1.1/tutorials/introductory/pyplot.html>`_
